# Hip Hop Fest 2020
- GraphQL API built using AWS AppSync, Amplify, DynamoDB and React.
Lets Users search Events, Stages, and Apps by name. 

![Hip Hop Fest 2020](Aloompa.gif)

## Technologies Used

-   React
-   GraphQL
-   AWS AppSync
-   Amplify
-   DynamoDB
-   SCSS
